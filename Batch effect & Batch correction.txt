Batch effect & Batch correction

Batch effect: 
실험의 실행 일련의 과정에서 발생하는 인위적인 변동으로, 샘플들이 어떤 기술적 요인에 의해 영향을 받아 생겨나는 문제입니다. 
예를 들어, 실험 일련의 과정에서 다른 시기에 실험을 수행하거나, 다른 장비나 연구자가 작업을 수행할 경우에 발생할 수 있습니다. 
이러한 batch effect가 발생하면 생물학적 차이와 혼돈되어, 실험 결과에 대한 해석에 오류가 발생할 수 있습니다.

Batch correction:
이러한 batch effect를 제거하거나 최소화하는 기술로, 데이터를 다시 처리하여 각 batch에서 발생한 차이를 제거하는 것입니다. 
Batch correction은 실험의 일관성을 향상시켜, 다른 batch 간에 비교 가능한 결과를 얻을 수 있습니다. 
Batch correction은 실험 데이터의 전처리 과정 중 하나로, 다양한 방법이 존재합니다. 가장 일반적인 방법으로는 각 batch에서의 
데이터 분포의 차이를 파악한 뒤, 이를 보정하는 방법이 있습니다. 다른 방법으로는 ComBat, sva 등의 특수한 알고리즘을 사용하여 batch effect를 보정하는 방법도 있습니다
