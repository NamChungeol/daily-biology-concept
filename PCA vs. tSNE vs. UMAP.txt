PCA vs. tSNE vs. UMAP

PCA(Principal Component Analysis): 다변량 데이터의 차원을 축소하거나, 상호 관계를 파악하는 데에 널리 사용되는 방법 중 하나입니다. 데이터를 새로운 축으로 변환하여, 
                                   주요 정보를 유지하면서 차원을 축소하는 기술입니다.
                                   
PCA는 데이터의 분산을 최대한 보존하는 방향으로 주요 성분을 추출합니다. 이렇게 추출된 주요 성분은 서로 직교하며, 데이터의 분산이 가장 큰 방향으로 정렬됩니다.
이렇게 주요 성분을 추출한 후, 이를 이용하여 차원 축소를 수행할 수 있습니다.PCA는 다양한 분야에서 활용됩니다. 
예를 들어, 고차원 데이터에서 중요한 성분을 추출하여 데이터를 시각화하거나, 데이터의 노이즈를 제거하거나, 데이터의 상호 관계를 파악하는 데에 활용됩니다. 
또한, PCA는 이미지 처리, 음성 인식, 유전체 분석 등 다양한 분야에서도 사용됩니다.



t-SNE는 데이터 시각화 알고리즘 중 하나로, 고차원 데이터를 저차원 공간으로 매핑하여 시각적으로 표현하는 방법입니다.
t-SNE는 "t-distributed Stochastic Neighbor Embedding"의 약자로, 주어진 데이터셋의 특성을 유지하면서 고차원 데이터를 2차원 또는 3차원 공간으로 축소하는 알고리즘입니다.
각 데이터 포인트 사이의 유사도 계산: t-SNE는 먼저 데이터 포인트 간의 유사도를 측정합니다. 이는 거리 기반 또는 유사도 기반으로 수행될 수 있습니다.

고차원 공간에서의 데이터 포인트의 확률 분포 계산: t-SNE는 이제 고차원 공간에서 각 데이터 포인트의 확률 분포를 계산합니다. 
이 확률 분포는 데이터 포인트 간의 유사도를 기반으로 계산됩니다.

저차원 공간에서의 데이터 포인트의 확률 분포 계산: t-SNE는 이제 저차원 공간에서 각 데이터 포인트의 확률 분포를 계산합니다. 
이 확률 분포는 높은 차원에서의 데이터 포인트 간의 유사도를 기반으로 계산됩니다.

KL divergence 최소화: t-SNE는 고차원 공간과 저차원 공간 간의 차이를 최소화하도록 노력합니다. 이를 위해 KL divergence를 최소화하는 방향으로 데이터 포인트를 이동시킵니다.

t-SNE는 시각적으로 매우 효과적인 방법으로, 데이터셋에서 패턴을 발견하는 데 유용합니다. 특히, 데이터셋 내에서 비슷한 데이터 포인트는 서로 가깝게 매핑되어, 
클러스터를 시각적으로 파악하기 쉽게 만들어줍니다. 따라서 t-SNE는 특히 생물학 연구 분야에서 유전자 발현 분석이나 단일 세포 분석 등에 활용되며, 
데이터 시각화와 탐색적 데이터 분석에 매우 유용합니다.




UMAP은 "Uniform Manifold Approximation and Projection"의 약자로, 고차원 데이터를 저차원 공간으로 매핑하여 시각화하는 머신러닝 기술 중 하나입니다. 
t-SNE와 마찬가지로, UMAP은 비선형 차원 축소 알고리즘으로, 대량의 고차원 데이터를 시각적으로 분석하기 위한 방법으로 사용됩니다.

UMAP은 주어진 데이터셋의 구조를 보존하면서 저차원으로 축소하는 것을 목표로 합니다. t-SNE와 달리, UMAP은 정확하고 빠른 알고리즘으로 알려져 있습니다.
각 데이터 포인트 사이의 근접성 측정: UMAP은 먼저 주어진 데이터셋에서 각 데이터 포인트의 근접성을 측정합니다. 이는 이웃 그래프를 만들어 계산됩니다.

근접성을 최대한 보존하면서 저차원 공간으로 매핑: UMAP은 이제 주어진 데이터셋을 저차원 공간으로 매핑합니다. 이 때, 근접성을 최대한 보존하는 것을 목표로 하며, 
최적화 알고리즘을 사용하여 저차원에서의 데이터 포인트의 위치를 조정합니다. UMAP은 데이터셋의 구조를 시각화하기 위해 사용됩니다. 
특히, t-SNE와 마찬가지로, UMAP은 대규모 데이터셋에서의 시각화에 매우 유용합니다. 




PCA, t-SNE, UMAP은 모두 차원 축소 방법 중 하나이지만, 다음과 같은 차이점이 있습니다.
1. 원리 :
PCA는 데이터의 분산을 최대한 보존하는 방향으로 주요 성분을 추출합니다. t-SNE는 고차원 공간에서 가까운 데이터들을 저차원 공간에서도 가깝게 유지하며, 
멀리 떨어진 데이터들은 멀리 배치합니다. UMAP도 고차원 공간에서 가까운 데이터들을 저차원 공간에서도 가깝게 유지하며, 멀리 떨어진 데이터들은 멀리 배치하지만, 
t-SNE보다 계산 속도가 빠릅니다.

2. 성능: 
PCA는 선형 차원 축소 방법으로, 대부분의 경우 데이터의 분산을 잘 보존할 수 있지만, 비선형 데이터에서는 제대로 작동하지 않을 수 있습니다. 
t-SNE와 UMAP은 비선형 차원 축소 방법으로, 비선형 데이터에서 잘 작동합니다. 그러나, t-SNE는 과잉적합(overfitting)되기 쉬우며, UMAP은 
시각화 결과에 대한 해석이 어렵다는 한계가 있습니다.

3. 계산:
속도 PCA는 계산 속도가 빠르며, 대용량 데이터에도 적용이 가능합니다. t-SNE와 UMAP은 계산 속도가 느리며, 대용량 데이터에 적용하기 어렵습니다. 
그러나, UMAP은 t-SNE보다 계산 속도가 빠릅니다.
따라서, PCA는 대용량 데이터에서도 적용이 가능하고, 선형 데이터에서 잘 작동하지만, 비선형 데이터에서는 한계가 있습니다. 
t-SNE와 UMAP은 비선형 데이터에서 잘 작동하지만, 계산 속도가 느리고, 시각화 결과에 대한 해석이 어려울 수 있습니다. 
이러한 특성을 고려하여, 각각의 방법을 상황에 맞게 선택하여 사용해야 합니다.
